Fitcheck Privacy Policy

Last updated: May 2026

Fitcheck is a Chrome extension that gives you personalized sizing recommendations on fashion product pages. This policy explains what data the extension accesses and how it is handled.

Data we collect

Fitcheck does not collect, transmit, or store any data on external servers. All data stays on your device.

  • Your profile (usual sizes, fit preferences, optional body measurements) — stored only in chrome.storage.local on your machine.
  • Brand notes and fit history — stored only in chrome.storage.local.
  • Product page content — the extension reads DOM content from fashion product pages you visit in order to extract size charts, size options, and fit signals. This content is processed locally and is never transmitted to any Fitcheck server.

Local API

If you choose to run the optional local API (npm run api), it runs on your machine at 127.0.0.1:8787 and is not accessible from the internet. When enabled, the extension sends product data and your profile to this local process. The local API may call third-party services you configure:

  • Gemini (Google AI) — if you set a Gemini API key and enable model-assisted mode. Product data (brand, title, size chart) is sent to Google's API. Your personal measurements and profile are included in the prompt.
  • Firecrawl / Brave Search — if you provide a search API key and enable web evidence mode. Queries like "Brand sizing reddit" are sent to the respective search provider. No personal data is included in search queries.

All third-party API keys are stored in a .env file on your machine. They are never stored in the extension or transmitted by the extension itself.

Permissions used

  • activeTab — read the current tab's URL and send messages to the content script.
  • storage — save your profile, history, and brand notes locally.
  • tabs — detect whether the active tab is a product page.
  • host_permissions (all URLs) — inject the content script on product pages so Fitcheck can read size charts and size options from the DOM.

Data sharing

Fitcheck does not share, sell, or transmit your data to any third party, except as described above when you explicitly configure a third-party API key.

Data retention

All locally stored data (profile, history, brand notes) remains on your device until you clear it via the Fitcheck options page or uninstall the extension.

Contact

If you have privacy questions, open an issue on the project repository.

Fitcheck is open-source software. Your data is yours.
